# /public

Coloque aqui sua foto pessoal:

- **Nome do arquivo:** `bruno.jpg` (ou `bruno.png` / `bruno.webp` — basta ajustar o `src` em `index.html`)
- **Formato sugerido:** quadrada (1:1), 800×800px, fundo neutro
- **Peso:** abaixo de 300 KB (use [squoosh.app](https://squoosh.app) para otimizar)

Enquanto a foto não estiver aqui, o site mostra automaticamente um placeholder elegante com as iniciais **BC**. Layout não quebra.
